<b>Native ZFS for Linux!</b>  ZFS is an advanced file system and volume manager which until now has only been available for Solaris.  This Linux ZFS kernel port currently includes a fully functional and stable SPA, DMU, and ZVOL with a ZFS Posix Layer (ZPL) on the way!
