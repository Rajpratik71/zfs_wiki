Native ZFS for Linux has been sanity tested using the ztest, zconfig, and zpios-sanity regression tests on the following platforms.  While this package builds on all of these platforms, the x86 platform is not as well tested and you are likely to encounter "problems":http://wiki.github.com/behlendorf/zfs/faq#fn1.  I _strongly_ suggest you stick to a x86_64 based platform for now.  Please feel free to add the results for your platform of choice.
